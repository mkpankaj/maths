# This is an eductional application to teach maths to kids. Application will explain concepts and then provide quiz.

## Maths Topics
•	Factors and Multiples
•	Fractions and Operations
•	Decimal System
•	Percentages
•	Numbers
•	Shapes and Measurements

## Workflow
1. User opens the application.
2. User selects the topic. User will see option to Take Quiz or Read the Subject. 
3. User clicks Read the Subject. Show all the topics within the subject and provide link to YouTube videos to understand the topics. 
4. User clicks Take Quiz.
5. Quiz will be of 10 questions with one question at a time on screen. After user submits the answer, next question appears on screen.
6. Show the marks at the end of quiz.
7. User can click a question to see explanation of correct answer.
8. As per marks, reward the user with funny emoji / caricature to keep him motivated.
9. Build a question bank of 50 questions for each topic. These will be randomly used for each quiz.
10. Keep refreshing the question bank with new questions.

## Technology Stack
1. react, react-dom:    UI framework
2. react-router-dom: 	Client-side routing
3. vite, typescript: 	Build tooling
4. tailwindcss: 	Styling
5. canvas-confetti:	Confetti on high scores
6. react-katex, katex:	Maths equation rendering
7. idb-keyval:	Settings & progress storage
8. zustand:	Quiz session state
